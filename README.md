## 대형병원 간호사 인수인계 앱

### 구조
1. 로그인
2. 첫 번째 탭(메인화면, 캘린더, 근무자 리스트) - 누르면 근무자 상세 액티비티
3. 두 번째 탭(인수인계 작성)
4. 세 번째 탭(인수인계 리스트) - 누르면 인수인계 상세 액티비티


### 역할분담
#### 김세원
- 메뉴(프로필, 로그아웃 버튼)
- 로그인 액티비티
- DB(근무자, 환자, 근무지)
=> 인수인계 작성 액티비티로 넘어가기

#### 김주희
- 인수인계 작성 액티비티(인수인계 뷰 생성)
- 인수인계 조회 액티비티(리스트, 상세 2개)

#### 김재현
- 메인화면 구성
- 근무자 조회 액티비티(리스트, 상세 2개)
=> 인수인계 조회 액티비티로 넘어가기
